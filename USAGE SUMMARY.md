## 🎯 Usage Summary

```bash
# 1. Make script executable
chmod +x install_b4j_wine.sh

# 2. Run installer (follow terminal prompts)
./install_b4j_wine.sh

# 3. Launch B4J after installation:
#    • From Application Menu → Search "B4J"
#    • Or from Desktop icon
#    • Or via terminal:
WINEPREFIX="$HOME/.wine_b4j" wine "C:\\Program Files\\Anywhere Software\\B4J\\B4J.exe"
```
